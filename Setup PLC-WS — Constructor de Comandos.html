Linealización de cisterna y control de script
SCL — tres tramos lineales · SSCR — arranque y parada del script asociado
▶
Linealización — SCL
Modelo: y = mₙ·x + bₙ  ·  Tramo 1: 0 → n1  ·  Tramo 2: n1 → n2  ·  Tramo 3: n2 → tope
Puntos de corte
n1 — fin tramo 1
valor del sensor en el primer corte
n2 — fin tramo 2
n2 debe ser mayor que n1
Tramo 1  0 → n1
m1 — pendiente
b1 — ordenada al origen
Tramo 2  n1 → n2
m2 — pendiente
b2 — ordenada al origen
Tramo 3  n2 → tope
m3 — pendiente
b3 — ordenada al origen
Comando SCL
>SCL1800,7400,1242,270990,2032,1611400,149,2235700<

▶
📶
Actualización por WiFi
Requiere red websat · FW v8.3+
▶
1
Consultar versión de firmware actual
>QVR<
2
Habilitar conexión WiFi
Crear hotspot desde celular o router con estos datos:
SSID
websat
Contraseña
12345678
3
Enviar comando de actualización por WiFi
El equipo no responde SMS. Monitorear por consola UART0.
>SFRMM1<
4
Esperar 2 minutos
Aguardar que el equipo descargue, flashee y reinicie.
5
Enviar factory reset
⚠ Irreversible — borra toda la configuración y restaura valores de fábrica.
>SFACTORY<
6
Verificar nueva versión de firmware
✓ Confirmar que la versión cambió respecto al paso 1.
>QVR<

Referencia rápida de comandos
Consulta, control y sistema
▶
Consultas — Q (responden con SMS)
>QGPS<Posición GPS + mapa
>QIO<Estado I/O completo (4 SMS)
>QNET<Estado de red e IPs
>QMDM<Info del módem
>QVR<Versión firmware + OTA
>QAIxx<Config canal analógico xx
>QCL<Consulta linealización cisterna
>QTD<Consulta intervalo RCQ
>QKA<Consulta intervalo Keep-Alive
>QTMRx<Consulta valor timer x
>QIP2<Consulta URL firmware FOTA
>QAPWn<Consulta red WiFi n → RAPWn...
>QSCRn<Consulta estado script n
>QCQ<Trama RCQ (UDP)
>QHELP<Ayuda de comandos
Seteos — S (sin respuesta SMS)
>SIPx"host",pto<IP destino x
>SIP2"url"<URL firmware FOTA
>SCXAPN...<Configuración APN
>SAPWaa...<Config red WiFi aa (SSID, pass, IP)
>SAIaa,...<Config canal analógico aa
>SCLn1,n2,...<Linealización cisterna
>STDxxx<Intervalo RCQ (30–3600 s)
>SKAxxx<Keep-alive backup (15–3600 s)
>STMRx:n<Timer x = n min (solo T1 y T2)
>SSCRn:ON/OFF<Arrancar / parar script n
>SRESET<Reiniciar equipo
>SFRMM1<Actualización OTA por WiFi (FW v8.3+)
>SFRMM2<Actualización OTA por Módem (FW v9.3+)
>SFACTORY<⚠ Factory reset — irreversible

Los comandos de Seteo (S...) no envían respuesta SMS. Verificar con el comando Q correspondiente luego de aplicar.
